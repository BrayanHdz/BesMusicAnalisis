Privacy Information

Bes Music does not collect any data. However it's based on the OneDrive API. For all Interaction with OneDrive you will also need to check OneDrives Privicy information.

  • Access your Music Library

    Bes Music normaly stores the downloaded Music in its app folder. So when the app is removed so is the no longer used Music. You can instead choose to store the downloaded Files in your Music folder. This allows you to store the Music on a different storage then the app (important for Mobile with limited internal storage). In addition it will not be deleted on uninstall.

  • Access Internet (Client)

    Bes Music is a client that relies havely on OneDrive. So we need to access the OneDrive API. So we need Internat. We should not do any other requests. If you notice something please open a Bug.

  • Read Access to OneDrive

    Bes Music is a client that relies havely on OneDrive. So we need to access the OneDrive API. We only have read access. so we will not accedently remove any of your music.

  • Read/Write access to AppFolder on OneDrive

    We use this Folder to syncronize your playlists. Since is it isolated for this App it should not interefere with any other app.
